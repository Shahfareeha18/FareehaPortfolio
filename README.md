# Syeda Fareeha Portfolio Next.js Template

A comprehensive Next.js template built with TypeScript, featuring modern development tools, responsive design, and extensive customization options.

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **View documentation:**
   ```bash
   npm run docs
   ```

## Documentation

This template includes comprehensive documentation covering all aspects of customization and development. Launch the interactive documentation with:

```bash
npm run docs
```

The documentation covers:
- **Getting Started** - Installation, setup, and first steps
- **Customization Guide** - Logo, favicon, and theme customization
- **Routing & Pages** - Adding pages with Next.js App Router
- **Layout System** - Header, footer, and layout customization
- **Data Management** - Working with JSON data files
- **Components Guide** - Understanding and creating components
- **Loading & Performance** - Loading states and optimization
- **Styling Guide** - Tailwind CSS and custom styling
- **Deployment** - Building and deploying your Next.js app
- **Troubleshooting** - Common issues and solutions
